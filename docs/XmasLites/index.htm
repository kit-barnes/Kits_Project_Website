XMAS LITES

Late in 2018, my wife and I decided we had to replace our ancient Christmas tree lights. I definitely wanted LEDs and, as long as I was about it, individually addressable ones. I ordered a couple of WS2811 strings, a power supply, and a "14keys Wireless RF Led RGB Controller". While that controller had an impressive array of routines to select from and even allowed some customization, I was not happy with the patterns available - thus this project.

Because I had promised my wife the tree would be up with the lights on it by Thanksgiving (she wanted a LOT of time for decorating), I expected to still be working on the code while my controller was buried in the tree. Consequently, I chose the NodeMCU platform with FTP and telnet servers to facilitate remote development of the software.

Schematic (with stocklist), source, and instructions are available at:

https://github.com/kit-barnes/XmasLites

The code is pretty much what was in place for Christmas. The tree, LED strings, and controller are stored away for next season. Sometime around Thanksgiving, I expect to be updating it again.
